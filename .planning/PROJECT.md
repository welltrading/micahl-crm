# מיכל CRM — דאשבורד ניהול קמפיינים ולקוחות

## What This Is

דאשבורד ווב למיכל — מאמנת עסקית שמלמדת נשים עם קליניקות לייצר לקוחות.
המערכת מחליפה תהליך ידני של הקמת אוטומציה לכל אירוע (רב מסר + MAKE.com + Google Sheets + App Script) בממשק אחד פשוט לניהול קמפיינים, תזמון הודעות ווצאפ, ו-CRM בסיסי.

## Core Value

מיכל יוצרת קמפיין לאירוע חדש בדקות — בעצמה, ללא עזרת איש אוטומציה — מגדירה תאריך ותוכן, והמערכת שולחת את כל הודעות הווצאפ בזמן הנכון לכל הנרשמות.

### Out of Scope

- הודעות מותאמות אישית לכל לקוחה — כל ההודעות זהות לכולן, לא כדאי לסבך v1
- ניהול תשלומים / כספים — לא חלק מהמשפך הנוכחי
- אפליקציית מובייל ייעודית — ווב responsive מספיק
- חיבור ישיר לרב מסר לניהול רשימות — רק ייבוא, לא ניהול מלא
- MAKE.com / Google Sheets אינטגרציה בעתיד — המערכת מחליפה אותם, לא מתחברת אליהם

## Context

**המצב הנוכחי:**
- לידים מגיעים מדפי נחיתה של רב מסר
- MAKE.com מטריג שליחת הודעות לפי נתונים ב-Google Sheets
- App Script מנהל את התזמון
- מיכל מקימה את כל האוטומציה מחדש לכל אירוע — תהליך שלוקח שעות

**נפח:**
- 1-2 אירועים בחודש
- מיכל היא המשתמשת היחידה (אין צוות)

**אינטגרציות קיימות:**
- GREEN API: שליחת הודעות ווצאפ (API key קיים)
- רב מסר: מקור הלידים (ייבוא נדרש)

## Constraints

- **משתמש יחיד**: רק מיכל — אין צורך ב-multi-tenancy או permissions מורכב
- **ווצאפ בלבד**: ערוץ התקשורת היחיד הוא GREEN API ווצאפ
- **עברית**: כל הממשק בעברית (RTL)
- **אמינות שליחה**: הודעות שתוזמנו חייבות לצאת בזמן — זה הליבה
- **Airtable כ-database**: כל הנתונים (לקוחות, קמפיינים, הודעות) יושבים ב-Airtable — מאפשר למיכל לראות ולערוך ישירות
