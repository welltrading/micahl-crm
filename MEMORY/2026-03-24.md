## מה עדיין פתוח

### Make — צריך לבנות
1. **Broadcast Scenario**
   - Trigger: Webhook מ-CRM
   - קלט: `{ campaign_id, message_content, target: "enrolled"|"interested"|"both" }`
   - לוגיקה: לפי target, שלוף מ-`נרשמות` / `מתענינות` / שתיהן
   - שלח GREEN API לכל נרשמת
   - (אין עדכון סטטוס נדרש לbroadcast)

2. **Scheduled Messages Scenario**
   - Trigger: Schedule כל 15 דקות
   - שלוף מ-`הודעות מתוזמנות`: תאריך שליחה = היום + סטטוס = ממתינה
   - סנן: שעת השליחה <= שעה נוכחית בישראל
   - שלוף נרשמות לקמפיין מ-`נרשמות`
   - שלוף טלפון מ-`אנשי קשר`
   - שלח GREEN API
   - עדכן סטטוס → נשלחה
